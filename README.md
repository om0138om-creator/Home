## العربية

### ✨ المميزات

#### 🔤 دعم OpenType الكامل
- **Standard Ligatures (liga)** - الرباطات القياسية
- **Discretionary Ligatures (dlig)** - الرباطات الاختيارية
- **Contextual Alternates (calt)** - البدائل السياقية
- **Stylistic Alternates (salt)** - البدائل الأسلوبية
- **Swash (swsh)** - الزخرفات
- **Small Capitals (smcp)** - الأحرف الكبيرة الصغيرة
- **Oldstyle Figures (onum)** - الأرقام القديمة
- **Lining Figures (lnum)** - الأرقام المصفوفة
- **Tabular Figures (tnum)** - الأرقام الجدولية
- **Fractions (frac)** - الكسور
- **Superscript/Subscript (sups/subs)** - الأرقام العلوية والسفلية
- **Kerning (kern)** - المسافات بين الحروف
- **Stylistic Sets (SS01-SS20)** - المجموعات الأسلوبية
- **Character Variants (CV01-CV99)** - متغيرات الأحرف

#### 🎨 تصميم احترافي
- واجهة عربية بالكامل
- دعم الوضع الداكن والفاتح
- تصميم متجاوب للموبايل والتابلت
- Touch Gestures (تكبير وتصغير وتحريك باللمس)

#### 📁 إدارة الخطوط
- إضافة خطوط متعددة دفعة واحدة
- دعم TTF, OTF, WOFF, WOFF2
- كشف تلقائي لمميزات OpenType
- البحث في الخطوط
- تعديل أسماء الخطوط
- حفظ الخطوط محلياً

#### 🖼️ التصدير
- PNG (شفاف)
- JPG (مضغوط)
- WebP (للويب)
- تحكم في الجودة

### 📥 التثبيت

#### من GitHub Releases
1. اذهب إلى [Releases](https://github.com/yourusername/font-studio-pro/releases)
2. حمّل أحدث APK
3. ثبّت على جهاز Android

#### بناء من المصدر
```bash

git clone https://github.com/yourusername/font-studio-pro.git
cd font-studio-pro

# تثبيت المتطلبات

# تشغيل محلياً
